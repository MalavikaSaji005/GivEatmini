# **App Name**: GivEat: Food Connect

## Core Features:

- Login & Guest Access: Welcome/Login Screen with options for login via phone/email or continue as guest.
- Home Navigation: Home Screen with clear navigation buttons: Donate Food, Sell Homemade Food, and Community Connect.
- Donation Listings: Donate Food Page displaying a list of nearby donated food with filter options.
- Post Donation Feature: Functionality to post a food donation with details like name, description, location, and expiry time.
- Homemade Food Marketplace: Sell Homemade Food Page to display available meals and allow users to post their own food items for sale, with integrated FSSAI support.
- Community Connection: Community Page featuring a list of NGOs, Shelters, and Local Volunteers with connection options.
- Voice Assisted Input: Add voice input via microphone to simplify form inputs throughout the app; the tool will reason whether to extract information and automatically populate the field.

## Style Guidelines:

- Primary color: Olive Green (#556B2F) to reflect nature, health, and a sense of community.
- Background color: Desaturated Olive, near Beige (#FDF6E3) for a clean, warm backdrop.
- Accent color: Yellow-Green (#9ACD32) to draw attention to calls to action like posting a new listing; this provides good contrast and falls close to the primary on the color wheel.
- Body and headline font: 'Poppins' (sans-serif) for a friendly, modern, and readable interface.
- Simple, line-based icons related to food, community, and locations.
- Clean, functional layout with subtle drop shadows and rounded edges; cards for listings; avoid gradients.
- Subtle animations to improve user engagement
